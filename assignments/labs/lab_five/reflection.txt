What is the purpose of the $_FILES superglobal in PHP?
It is the built in way PHP exposes information about files uploadted through an html form.

Why does a form need special settings to upload files? 
Because browsers and servers don't treat file data the same way they do normal text fields.

What function is used to move uploaded files to a folder?
The function used to move an uploaded file into a permanent folder is move_uploaded_file(). 

Why is it important to control where uploaded files are stored? 
Controlling where files are uploaded helps with sustanibility, security, and organization.
